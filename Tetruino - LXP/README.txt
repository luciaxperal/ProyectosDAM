Para abrir el proyecto y poder jugar, puedes usar el ejecutable. 
Se encuentra en la carpeta de Tetruino exe, y hay dos carpetas. 

	- La versión Tetruino es con el Arduino, aunque también funcionan las teclas. 
	- La versión Tetris, es la misma que la otra con la única diferencia de que todo el código que se necesita de Arduino no está. Es una versión que puede que funcione mejor ya que no tiene esos métodos del Arduino, ni tampoco sus Timer, por lo que el proceso será más ágil.

También en el código he subido las dos versiones, y el código en Arduino.
